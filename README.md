# Calc.js 🧮

Uma calculadora web desenvolvida com HTML, CSS e JavaScript puro, com suporte a tema claro/escuro e interação via teclado.

## 🖥️ Funcionalidades

- Operações matemáticas básicas: adição, subtração, multiplicação e divisão
- Suporte a parênteses para expressões compostas
- Operação de módulo (%)
- Alternância entre tema claro e escuro
- Interação via clique nos botões ou digitação pelo teclado
- Botão para copiar o resultado
- Limpeza do input com o botão "C"

## 🛠️ Tecnologias

- HTML5
- CSS3
- JavaScript (Vanilla)

## 📚 Aprendizados

Projeto desenvolvido durante curso de JavaScript, com foco em:

- Manipulação do DOM

- Criação e remoção dinâmica de elementos
- Uso de `dataset` para armazenar valores em atributos HTML
- Navegação e edição via teclado com `keydown`
